Что сделано:
- Макет собран из смысловых секций, а не случайно нарезан.
- Каждая секция экспортирована из исходника и приведена к ширине 1280px.
- На сайте выглядит 1 в 1 с исходным макетом.
- Добавлена кликабельная зона на финальную кнопку «Начать путь».
- Добавлен скрытый SEO-текст.

Исходный размер: 1704x9552px
Итоговая ширина сайта: 1280px
Итоговая высота полного макета: 7175px

Как загрузить:
1. Распакуй архив.
2. Загрузи index.html, папку css и папку assets в корень хостинга.
3. Открой сайт.
